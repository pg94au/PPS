The PPS printing system was originally developed for the Linux operating system as a replacement for the BSD lpr printing system. The goals in creating this system were ease of management and use, as well as built-in page accounting facilities. It was created for use in the Faculty of Education at Brock University, where it is being used as a means by which laser printing services are provided to students and staff in a public computer lab.

NOTE: I have not been maintaining this project since August 11, 2000.


Features

  • Built in support for page counting and charging
  • Easy administration via World Wide Web interface (see here)
  • Integrated database to maintain user balances
  • Can easily be used with samba to provide services to Windows clients
  • Familiar lpr-like user interface


Requirements

  • Tested on Linux (2.0.34), Solaris 7 (gcc 2.8.1), HPUX 10.20 (gcc 2.95.2), IRIX 6.5 (gcc 2.8.1)
  • Builds (but untested) under IRIX 5.3
  • PJL compatible laser printer w/ JetDirect compatible interface
    (this includes all HP LaserJet model 4 or newer printers with an ethernet connection)
  • Mini SQL database server (must be obtained separately)
  • Web server with CGI capability (the Apache server can be obtained freely)
