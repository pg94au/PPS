PPS REVISION HISTORY


08/11/2000

  • Port to build under IRIX 5.3 and IRIX 6.5.

05/08/2000

  • Fix core dump under HPUX 10.20 after each job prints.

04/12/2000

  • Port to build under HPUX 10.20.
  • Fix bug in 'make installman'

06/20/1999

  • Support for Solaris 2.7 added. Configuration now done using autoconf, which means untested platforms may work already as well. Experimental support for IRIX 6.5 (compiles, but not tested).

02/09/1999

  • Added support for an HTML based form through which users can check their current print balances.

02/02/1999

  • Created a new admin_log file in the logs directory, to which changes made to user balances are logged.

01/06/1999

  • Added code, to enforce an upper limit on log file sizes, which automatically trims logs.
  • Fixed a problem which didn't affect page counting, but logged some cancelled jobs as having fully completed.

11/27/1998

  • First public release.
