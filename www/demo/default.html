PPS ADMINISTRATION

The following are screen captures taken from the administration utilities for the PPS printing system, as viewed through Netscape.



The queue/printer setup menu displays all queues and their attached printers currently in the system. Each queue or printer can be viewed by selecting it, and new queues or printers can be added by selecting the appropriate link.



The queue display/update form allows you to set the various attributes for each queue in the system, including the method of charging and the amount to be used for page accounting. Valid methods are charging per page, per job, and having no charge.



When new printers are created, the type of printer and interface is chosen first, along with all of its device independent characteristics.



Once the device and its interface have been selected, information pertinent to the specific device configuration is requested.



In addition to device management, users of the printing system are also managed through a similar interface. Users can be added, or viewed, allowing them to be modified.



Each user in the system has essential information, such as a usercode, balance and accounting status, as well as information useful for administrative purposes, such as a full name, student id and other notes.
